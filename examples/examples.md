# examples

## 目录职责

本目录放可复制到 `~/.spotAgent/` 的本地配置示例，覆盖 Settings 中的 Plugin、Append Prompt 与 MCP 管理页面。

## 子节点

- `plugins/plugins.md`：Action manifest 示例目录，包含 Plugin 与 Append Prompt 两类。
- `mcp/mcp.md`：MCP server 配置示例目录。

## 使用方式

- Plugin 示例复制到 `~/.spotAgent/plugins/<plugin-id>/plugin.json`。
- MCP 示例复制到 `~/.spotAgent/mcp.json`，保存后重启桌面 App。
- Settings 页面也提供"添加示例"按钮，会写入同等结构的本地配置。
